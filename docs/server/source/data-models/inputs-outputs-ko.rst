입력 및 출력
==================

다음 링크를 사용하여 사용자와 관련된 BigchainDB Transactions Spec (또는 Specs)을 찾으십시오:

`BigchainDB 트랜잭션 스펙 <https://github.com/bigchaindb/BEPs/tree/master/tx-specs/>`_

그 다음 **트랜잭션 구성요소: 입력** 과 **트랜잭션 구성요소: 출력** 을 참조하십시오.
